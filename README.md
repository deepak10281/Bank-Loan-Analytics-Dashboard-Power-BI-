# Bank-Loan-Analytics-Dashboard-Power-BI-

An interactive and professional **Bank Loan Analytics Dashboard** built using **Power BI** to analyze loan portfolio performance, customer repayment behavior, lending operations, and financial risk metrics.

---

## 📌 Project Overview

This project focuses on analyzing a bank's loan dataset to generate meaningful business insights through powerful visualizations and KPIs.

The dashboard helps stakeholders:

✔ Monitor lending performance  
✔ Track good vs bad loans  
✔ Analyze repayment behavior  
✔ Measure financial risk  
✔ Understand borrower profiles  
✔ Make data-driven lending decisions  

---

# 🎯 Problem Statement

The objective of this project is to:

- Analyze loan portfolio performance and lending operations
- Track Good Loans vs Bad Loans
- Monitor repayment behavior
- Measure KPIs like:
  - Total Funded Amount
  - Total Amount Received
  - Interest Rate
  - Debt-to-Income Ratio (DTI)
- Build an interactive Power BI dashboard for business insights
